# 🃏 Descartez

**Descartez** é um jogo multiplayer online baseado em Uno.
## ✨ Funcionalidades

- [x] Graficos 2D
- [x] Multiplayer 
## 🛠️ Tecnologias Utilizadas

- React.js
- TypeScript
- PixiJs (WebGL)
## 🚀 Como rodar localmente

### Pré-requisitos:
- Node.js

### Instalação:
```bash
git clone https://github.com/AllyssinXD/card_game_web/card_game_web.git
cd card_game_web

# Instale as dependências
npm install

# Rode o servidor
npm run dev
```
